# 🖼️ Image Processing Fundamentals with OpenCV

This repository contains a structured collection of tutorials and notebooks covering the fundamentals of image processing using Python and OpenCV.

Each folder focuses on a specific core concept, starting from basic image structure to more advanced topics like color spaces and histograms.

---

## 📚 Course Structure

### 01 - Introduction: Image Structure & Types
Basics of digital images, pixel representation, and image types.

### 02 - OpenCV Image Basics
Loading, displaying, and handling images using OpenCV.

### 03 - Bit Depth
Understanding bit depth and how it affects image quality and color representation.

### 04 - Image Transparency
Working with alpha channels and transparency in images.

### 05 - Data Types (dtype)
Image data types in NumPy and their impact on image processing operations.

### 06 - Contrast & Histogram
Image contrast, histogram analysis, and intensity distribution.

### 07 - HSV Color Space
Understanding HSV (Hue, Saturation, Value) and its advantages over RGB in image processing tasks.

---

- Matplotlib
