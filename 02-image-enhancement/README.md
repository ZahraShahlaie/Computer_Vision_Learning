# ✨ Image Enhancement with OpenCV

This repository contains a structured collection of tutorials and notebooks covering the fundamental techniques of image enhancement using Python and OpenCV.

Each folder focuses on a specific enhancement technique, starting from brightness and contrast adjustment to edge detection, Hough Transform, and a real-world computer vision project.

---

## 📚 Course Structure

### 01 - Brightness & Contrast

Learn how image intensity affects visual quality and explore different techniques for enhancing image contrast.

Topics include:

- Histogram Stretching
- Histogram Equalization
- Limitations of Histogram Equalization
- CLAHE (Contrast Limited Adaptive Histogram Equalization)

---

### 02 - Filter & Noise

Understand different filtering techniques and how they are used to remove image noise while preserving important image details.

Topics include:

- Mean Filter
- Gaussian Filter
- Median Filter
- Bilateral Filter
- Image Noise

---

### 03 - Sharpening

Learn how sharpening enhances image details using spatial filtering and convolution techniques.

---

### 04 - Edge Detection

Study how object boundaries can be extracted from images using image gradients and edge detection algorithms.

Topics include:

- Image Gradients
- Sobel Operator
- Prewitt Operator
- Canny Edge Detector

---

### 05 - Hough Transform

Learn how geometric shapes can be detected using the Hough Transform.

Topics include:

- Hough Line Transform
- Hough Circle Transform

---

### 06 - Project: Parking Spot Detection

Apply the concepts learned throughout this chapter to build a complete computer vision project.

The project demonstrates a classical image processing pipeline for detecting parking spaces using:

- Image Preprocessing
- Edge Detection
- Hough Line Transform
- Line Filtering
- Parking Slot Detection

---

## 🛠️ Requirements

- Python 3.x
- OpenCV
- NumPy
- Matplotlib

---

## 🚀 Goal

This repository is designed to build a strong understanding of classical image enhancement techniques and their applications in computer vision.

By completing this section, you will gain practical experience in improving image quality, extracting meaningful features, and solving real-world vision problems using traditional computer vision methods.
