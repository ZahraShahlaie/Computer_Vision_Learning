# 🧠 Deep Learning for Computer Vision

This repository contains a structured collection of tutorials and notebooks covering the fundamental concepts of deep learning for computer vision using TensorFlow, Keras, and PyTorch.

Each folder focuses on a specific deep learning concept, starting from building neural networks with Keras to transfer learning, custom convolutional neural networks, and PyTorch fundamentals.

---

## 📚 Course Structure

### 01 - Transfer Learning & Fine-Tuning

Learn how to leverage pre-trained deep learning models to solve computer vision tasks efficiently.

Topics include:

- Transfer Learning
- Feature Extraction
- Fine-Tuning
- Pre-trained Models

---

### 02 - Keras Functional API

Learn how to build flexible and complex deep learning architectures using the Functional API in Keras.

Topics include:

- Functional API
- Multi-layer Networks
- Multiple Inputs & Outputs
- Model Construction
- Model Visualization

---

### 03 - Comparing Custom CNN and Transfer Learning

Explore the differences between designing a Convolutional Neural Network (CNN) from scratch and using Transfer Learning with pre-trained models.

Topics include:

- Custom CNN
- Image Classification
- Transfer Learning
- Performance Comparison
- Model Evaluation

---

### 04 - PyTorch Basics

Learn the fundamentals of deep learning development with PyTorch.

Topics include:

- Tensors
- Dataset & DataLoader
- Building Neural Networks
- Training and Evaluation Workflow

---

## 🛠️ Requirements

- Python 3.x
- TensorFlow / Keras
- PyTorch
- NumPy
- Matplotlib
- OpenCV

---

## 🚀 Goal

This repository is designed to build a strong foundation in deep learning techniques for computer vision.

By completing this section, you will gain practical experience in designing neural networks, applying transfer learning, training convolutional neural networks, and using modern deep learning frameworks for real-world computer vision applications.
